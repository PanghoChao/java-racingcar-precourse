# 자동차 경주게임 만들기 
## precourseWeek2


## 프로세스
 1. n대의 자동차 -사용자가  n개의 자동차 이름(5자이하) 각 이름은 쉼표(,)로 구분 입력
 2. 자동차 갯수에 맞춰  class car를 통해 인스턴스를 배열에 생성
 2. 사용자 이동할 횟수 입력 
 3. 0~9무작위 값이 나오며, 4이상만 전진
 4. 이동할 횟수만큼 이동 후, 가장 멀리간 자동차들이 우승 
 5. 잘못된 값입력할경우 에러발생 후 다시 입력
 
## 구현할 기능 목록
#### main 
 1. 자동차 갯수에 맞춰 car 인스턴스 생성 하고 배열에 보관 -> carArray[]

 2. 시도할 횟수 만큼 input2번에서 만든 배열 요소전부 실행
     - 각 요소들은 car1,2번  반복

 3. 1번을  마치고 제일 많이 전진한 자동차의 리스트를 출력 
     - 제일 숫자 높은 carArray[] 인덱스를 모으고
     - 차 이름 출력

#### input
 1. 자동차 이름을 한줄로 입력 받을 것 -> carNameArray[]
     - 쉼표로 자동차 이름을 구별한다.
     - 띄어쓰기 제거해준다.trim
  
 2. 시도할 횟수 int로 입력 받기
     - 잘못된 입력값이 에러를 발생시키고 다시 입력을 하겠끔하는 함수
  
#### car
 1. 메소드 : 0~9무작위값 하나 출력하는 함수
     - Randoms를 사용하여 출력
 2. 메소드 : 4이상이면 position을 +1 증가시킨다
 3. 메소드 : 현재 position의 값을 출력
 4. 메소드 : 현재 position을 visual로 보여주는 함수
 5. 메소드 : 인스턴스에서 차이름을 출력
 
 
## 주의사항
 1. 함수의 길이가 15라인을 넘지않도록 한다. 또한, 한가지일만 하도록 구현한다.
 2. else 예약어를 쓰지않는다. 
 3. indent depth를 2까지만 허용한다. 
 4. 3항 연산자를 쓰지않는다. 
 5. 별도로 변경불가 안내가 없는 경우 파일 수정과 패키지이동을 자유롭게 할수있다.
